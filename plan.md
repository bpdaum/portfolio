# Project Plan: Data Science Portfolio

## Overview
This document outlines the plan to build a personal portfolio website for Brian Daum, Data Scientist.

## Objectives
- Showcase 6 data science projects.
- Provide a clear call-to-action to download the resume.
- Establish a professional online presence.

## Technical Approach
We will use standard web technologies to ensure performance, SEO, and longevity without unnecessary overhead.
- **HTML5** for structure.
- **CSS3** for styling (using CSS Variables for easy theming).
- **JavaScript** for lightweight interactivity.

## Design Concept
- **Visuals**: Clean lines, ample whitespace, modern typography.
- **Theme**: Professional (Blues, Greys, with an accent color like Teal or Coral).
- **Layout**: Single-page scrolling design.

## Implementation Steps
1.  **Setup**: Initialize project structure (folders, blank files).
2.  **Structure**: Write semantic HTML for all sections.
3.  **Style**: Apply global styles, typography, and layout (Grid/Flexbox).
4.  **Content**: Fill in placeholders for the 6 projects and resume.
5.  **Refine**: Add polish (animations, hover states) and ensure responsiveness.

## Timeline
- **Phase 1**: Structure & Layout (In Progress)
- **Phase 2**: Styling & Theming
- **Phase 3**: Content Integration
- **Phase 4**: Final Review

## Sections Breakdown
1.  **Header/Nav**: Name and Navigation links.
2.  **Hero**: Large introductory area.
3.  **Projects**: 2x3 Grid of project cards.
4.  **About**: Professional summary.
5.  **Resume**: Prominent download link.
6.  **Footer**: Contact info and social links.
